About Me

My journey into technology began with a fascination for how data drives decision-making in modern businesses. As a Computer Information Systems major at Monroe University, I've cultivated this interest into a focused career path toward data engineering and backend development.

During my Java Backend internship at Revature, I gained invaluable experience developing RESTful APIs using Java and Spring Boot. I am particularly drawn to data engineering because it combines my strengths in programming, database management, and analytical thinking. I enjoy the challenge of designing efficient data pipelines, optimizing query performance, and ensuring data integrity across systems.
